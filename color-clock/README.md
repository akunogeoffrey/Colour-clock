# üïê Color Clock

A dynamic digital clock built with React, Vite, and date-fns.

## Features
- Displays the current time (hours, minutes, seconds)
- Displays the current date (day, month, year)
- Updates every second in real time
- Styled with a dark gradient theme and neon colors

## Tech Stack
- React (with Vite)
- date-fns v2.30.0
- CSS3

## How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/color-clock.git
cd color-clock
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```

### 4. Open in browser
Visit: `http://localhost:5173`
